# Installation Steps

1) nextjs
2) tailwind css
3) stripe
4) next auth
5) firebase


## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
